Bazi

Designing for the Internet of Things - Advanced - Carnegie Mellon University | Spring 2017
Independent Project

Scope

Bazi was created with the goal of taking long-distance competitive play out of the purely digital realm and moving it into the physical realm in mind.

This product was developed with the objecctive of learning how to code a product from scratch - for that reason a lot of user research and user experiece considerations were monetarily put on hold to enable growth in terms of technical competency.

A schematic of the circuit created with these components is below. This circuit was soldered to a quarter-sized perma protoboard for the final prototype.

This circuit was powered by a regardable 3.7v Lithium Ion Polymer Battery with a 2500mAh capacity. This LiPo was plugged into a 1000 Rechargable Powerboost which the powered the circuit using a micro-USB cord.

Fabrication

The physical form of Bazi was fabricated from 1/8" milky white acrylic. The shape, a cuboctahedron, was cut from the acrylic using a .dxf file developed in Adobe Illustrator. This lasercut form was hot-glued.

Code

For demostration purposes, a follower and leader device were coded in order to simply the the code.

Device 1, the leader, was also coded to use a push-button start as opposed to a hall-effects interrupt as a start in order to test a different iteraction with the device.

Device 2, the follower, uses a hall-effects interrupt as planned as a start to verify that interaction as the desired the one.
